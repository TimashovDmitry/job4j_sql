# job4j_sql

SQL, реляционные базы данных, PostgreSQL.

## Темы

- создание таблиц и схем
- первичные и внешние ключи
- ограничения NOT NULL, UNIQUE, CHECK
- вставка, обновление, удаление
- выборка с фильтрацией и сортировкой
- группировка и агрегатные функции
- JOIN и подзапросы
- транзакции
- индексы
- проектирование схем

## Структура

Скрипты в папке `scripts`.
